# Experiment 68: Frequency Dictionary (HashMap)

## Problem Statement
Read a line of text. Use a `HashMap` to count the frequency of every word in the text. Print the unique words and their counts.

## Input Format
* A single line of text.

## Output Format
* `Word: Count` (in any order).

## Example 1

**Input:**
```text
hello world hello java
```


**Output:**
```text
hello: 2
world: 1
java: 1
```
